# Introduction to Ticos Documentation

Welcome to the official documentation for the Ticos platform. This site provides comprehensive guides, tutorials, and references to help you understand, integrate, and effectively utilize Ticos for building and deploying advanced Embodied AI agents.

## What is Ticos?

Ticos is a cloud-based platform designed to simplify the development, management, and deployment of Embodied AI. It offers a suite of tools and services that enable developers to create sophisticated agents capable of perception, interaction, and intelligent decision-making in various environments.

## How to Use This Documentation

This documentation is organized into several key sections to help you find the information you need:

-   **Getting Started**: If you are new to Ticos, this section provides a step-by-step guide to set up your environment and build your first agent.
-   **Agent Configuration**: Learn about the various aspects of configuring your Embodied Agents, including their instructions, skills, and advanced settings.
-   **Playground**: Discover how to use the Playground to test and interact with your agents in a simulated environment.
-   **Settings**: Understand the global settings available on the platform, such as face management, terminal models, and MCP server configurations.
-   **Terminal Ops**: Find information on managing and monitoring your terminal devices.
-   **API Reference**: Detailed documentation for the Ticos APIs.
-   **Realtime Protocols**: Information on the realtime communication protocols used by Ticos.

Navigate using the sidebar to explore these sections. If you have any questions or encounter issues, please refer to our support channels or community forums.

We are constantly working to improve Ticos and its documentation. We welcome your feedback!
