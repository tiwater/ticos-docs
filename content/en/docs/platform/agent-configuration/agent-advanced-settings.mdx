# Advanced Settings

The "Advanced Settings" module is used for managing critical and high-risk operations related to the agent.

## Key Features and Operations

### Delete Agent

- Permanent agent deletion can be executed on this page.
- The deletion operation is irreversible and will clear all related configurations, knowledge base items, and all data associated with the agent, which cannot be recovered.
- Before proceeding, please make sure to back up all necessary information or confirm that the agent is no longer needed.

### Operation Tips

- Deletion operations typically require secondary confirmation to prevent accidental operations.
- Please carefully verify the agent you are currently managing to ensure the operation is correct.

---

Please use the high-impact operations on this page with caution to avoid irreversible data loss.
