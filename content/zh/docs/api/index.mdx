# Ticos API 总览

欢迎来到 Ticos 平台 API 文档。我们的 API 旨在为您的终端和账户提供全面的控制和集成能力。

## API 分类

我们的 API 主要分为两个级别：

1.  **账户级 API：** 这些 API 用于管理账户范围内的资源和配置。身份验证通常需要账户 API 密钥。
    *   [终端管理](./api/account-level/terminal-management) - 管理终端的注册和配置。

2.  **终端级 API：** 这些 API 用于特定于单个终端的交互，例如管理人脸数据或实时操作。身份验证通常需要特定于终端的密钥。
    *   [人脸管理](./api/terminal-level/face-management) - 管理特定终端上的人脸识别数据。

## 身份验证

身份验证方法因 API 级别而异：
*   **账户级 API：** 通常使用 `账户 API 密钥` 作为承载令牌 (Bearer Token)。
*   **终端级 API：** 通常使用 `终端密钥` 作为承载令牌 (Bearer Token)。

有关详细的身份验证说明，请参阅每个 API 组的具体文档。

## 基础 URL

Ticos API 端点的基础 URL 为：
`https://api.ticos.ai/v1`


## API 版本控制

所有 API 端点都进行了版本控制。当前版本为 `v1`。

我们建议您查阅每个 API 组中的“最佳实践”部分，以获取有关安全性、错误处理和性能的指导。
