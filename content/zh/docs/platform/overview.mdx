# 平台概览

欢迎使用 Ticos Cloud！当您成功登录平台后，将首先进入“智能体（Agents）”管理页面。这里是您管理所有具身智能体的中心枢纽。

本篇文档将为您介绍平台的主界面和核心功能导航。

## “智能体”页面

这是您登录后的默认着陆页。此页面允许您集中查看、创建和配置您的智能体。

### 主要功能

- **智能体列表**：以卡片形式清晰展示您拥有的所有智能体，包括名称、更新状态和描述。
    - **配置（Configure）**：每个智能体卡片上都有一个“Configure”按钮，点击后可进入[智能体配置](./agent-configuration)，对智能体的大脑、知识库、技能和感知交互能力等进行详细设置。
- **创建新智能体（Create New Agent）**：通过专用的创建卡片，您可以轻松启动并配置一个新的智能体。

### 侧边导航栏

位于页面左侧的导航栏是您在平台中穿梭的主要工具，提供了对各项核心功能的快速访问：

- **Agents**：返回到当前的智能体管理页面。
- **Playground**：提供一个沙盒环境，用于测试和与您的智能体进行实时交互。
- **Terminal Ops**：用于管理和监控连接到云平台的终端设备。
- **Settings**：进行账户及平台的全局设置。
- **Documentation**：链接到 Ticos Cloud 的官方文档网站。

通过以上功能，您可以高效地构建、管理和部署您的具身智能体。
