# Ticos Cloud 平台说明

欢迎使用 Ticos Cloud 平台文档。本文档将帮助您了解和使用 Ticos Cloud 的各项功能。

## 主要功能模块

### 1. 智能体配置

- **智能体仪表盘**：查看和管理您的智能体实例
- **智能体模型**：配置智能体的基础模型和参数
- **智能体知识库**：管理智能体可访问的知识库内容
- **智能体技能**：配置和扩展智能体的各项能力
- **多模态能力**：
  - 视觉：图像识别与处理
  - 听觉：语音识别与理解
  - 语音：语音合成与交互
  - 表情：情感识别与表达（待开发）
  - 手势：手势识别与驱动（待开发）
- **发布变更**：发布和部署智能体配置
- **高级设置**：高级配置项

### 2. 功能体验

- 在线体验智能体的各项功能
- 测试和验证智能体配置
- 交互式演示和示例

### 3. 终端运维

- 终端设备管理
- 设备状态监控
- 远程维护和更新

### 4. 系统设置

- 人脸识别信息管理
- 终端型号管理
- MCP 服务配置

### 5. 账户设置

- 个人资料管理
- 使用情况统计
- API 密钥管理
- 账单信息

## 快速开始

1. 访问 [智能体配置](platform/agent-configuration) 创建您的第一个智能体
2. 通过 [终端型号管理](platform/settings/terminal-models) 创建您的终端型号
3. 通过 [终端运维](platform/terminal-ops) 来创建和管理您的设备
4. 在 [功能体验](platform/playground) 中测试智能体功能
5. 在 [账户设置](platform/account/profile) 中管理您的个人资料和 API 访问
