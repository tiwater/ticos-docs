# 视觉 (感知)

“视觉”模块用于配置智能体处理和解读视觉信息的方式。通过这些设置，您可以精确控制智能体的视觉能力。

## 配置选项

### 面部检测

- **面部检测**：启用后，智能体将能够检测其视野中是否有人脸。
- **面部识别**：启用后，智能体将能够识别其视野内的人脸是否属于人脸库中的特定个体。
- **管理人脸库**：点击此项可跳转至人脸库管理模块，进行人脸信息的增删改查。

如果面部检测成功，您可以在模型的系统指令中通过以下宏来引用检测到的面部信息。当视野中有多个人脸时，系统将默认使用最大的人脸信息。

| 宏指令                               | 说明                                                                 | 可能的值                                                        |
| ------------------------------------ | -------------------------------------------------------------------- | --------------------------------------------------------------- |
| `{{tiwater.vision.user_emotion}}`      | 用户的情绪 (英文)                                                    | `happy`, `sad`, `angry`, `surprised`, `fearful`, `disgusted`, `neutral` |
| `{{tiwater.vision.user_emotion.cn}}`   | 用户的情绪 (中文)                                                    | `快乐`, `伤心`, `生气`, `惊讶`, `害怕`, `厌恶`, `平静`             |
| `{{tiwater.vision.user_gender}}`       | 用户的性别 (英文)                                                    | `Man`, `Woman`                                                  |
| `{{tiwater.vision.user_gender.cn}}`    | 用户的性别 (中文)                                                    | `男`, `女`                                                      |
| `{{tiwater.vision.user_age}}`          | 用户的年龄 (阿拉伯数字)                                              | 例如 `28`                                                       |
| `{{tiwater.vision.user_age_subsection.cn}}` | 用户的年龄段                                                         | `少年`, `中年`, `老年`                                          |
| `{{tiwater.vision.user_name}}`         | 用户的姓名 (需提前录入人脸库)                                        | 例如 `张三`                                                     |

**注意**：当视野中没有检测到人脸，或特定属性（如姓名）未识别时，宏将返回默认值（如 `neutral`, `平静`, `unknown`, `未知`）。

### 物体检测

- **描述**：启用或禁用智能体检测和识别其视野内特定物体的能力。
- **物体列表**：启用后，系统将提供一个预定义的物体列表。您可以勾选希望智能体特别关注和识别的物体。

通过合理配置这些视觉设置，您可以确保智能体根据其任务要求，高效且准确地“看见”并理解其所处的环境。
