# 智能体配置

登录 cloud.ticos.ai 首页后默认显示的是您所拥有的智能体，您可以在这里创建并管理您的智能体。

本节将引导您完成在 Ticos Cloud 平台上配置具身智能体的各项设置。合理的配置对于定制智能体的功能、交互方式及其个性化表现至关重要。

智能体配置界面包含多个模块，分别对应智能体不同方面的设置。在选择了特定的智能体后，您可以通过左侧的导航栏切换并配置以下模块：

### 大脑

- **[智能体仪表盘](./agent-configuration/agent-dashboard.mdx)**：查看智能体的整体状态、关键指标和基本信息。
- **[智能体模型](./agent-configuration/agent-models.mdx)**：配置驱动智能体核心逻辑和能力的基础 AI 模型。
- **[智能体知识库](./agent-configuration/agent-knowledge.mdx)**：管理和编辑智能体在交互中可以参考和使用的知识内容。
- **[智能体技能](./agent-configuration/agent-skills.mdx)**：为智能体配置和启用特定的工具或能力，以完成特定任务。

### 感知与交互能力
以下模块用于定义智能体如何感知环境以及如何与用户进行多模态交互：
- **[视觉](./agent-configuration/agent-vision.mdx)**：配置智能体的图像理解和视觉输入处理能力。
- **[听觉](./agent-configuration/agent-hearing.mdx)**：配置智能体的声音识别和听觉输入处理能力。
- **[语音](./agent-configuration/agent-speech.mdx)**：设置智能体的语音合成（TTS）选项。
- **[表情](./agent-configuration/agent-emotion.mdx)**：定义智能体在交互中表达情感的方式。
- **[手势](./agent-configuration/agent-gesture.mdx)**：配置智能体对手势的识别能力。

### 其他配置项
- **[发布配置](./agent-configuration/agent-publish.mdx)**：在完成所有更改后，将配置发布到线上，使智能体生效。
- **[高级设置](./agent-configuration/agent-advanced-settings.mdx)**：进行智能体相关的其他高级操作，例如删除智能体等。

每个模块的具体配置方法和选项将在其对应的文档页面中详细说明。请点击上方链接或通过导航栏访问各个模块的详细文档。
