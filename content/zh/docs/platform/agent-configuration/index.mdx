# 具身智能体配置

本节提供了在 cloud.ticos.ai 平台上配置您的具身智能体的全面指南。正确的配置是定制智能体功能、个性化及交互风格的关键。

具身智能体配置界面分为几个主要模块，每个模块负责智能体设置的特定方面。选择智能体后，您可以使用左侧边栏菜单在这些模块之间导航。

以下是主要配置模块的概览：

- **[仪表盘](./agent-dashboard.mdx)**：提供智能体状态和关键信息的概览。
- **[指令](./agent-instructions)**：定义智能体的核心操作参数，包括其基本用途、如何感知环境（视觉、听觉）以及如何反应（语音、情绪、姿态、行为）。
- **[知识](./agent-knowledge.mdx)**：管理智能体用于提取信息和回答查询的知识库。
- **[技能](./agent-skills.mdx)**：配置智能体可以利用的工具和能力。
- **[形象](./agent-avatars.mdx)**：自定义智能体的外观和角色（链接到全局设置）。
- **[模型](./agent-models.mdx)**：管理驱动智能体各项功能的人工智能模型（链接到全局设置）。
- **[发布更改](./agent-publish.mdx)**：审查并部署智能体的配置更改。
- **[高级设置](./agent-advanced-settings.mdx)**：访问管理智能体本身的设置，例如删除智能体。

每个模块将在其各自的子章节中详细介绍。请参阅特定页面以获取深入的说明和功能描述。
