# 智能体仪表盘 (总览)

智能体仪表盘，也称为总览页面，是您选择智能体后看到的第一个屏幕。它提供了智能体基本信息的摘要以及对基本操作的快速访问。

## 显示的信息

仪表盘通常显示所选智能体的以下详细信息：

- **智能体名称**: 用户定义的智能体名称 (例如, "测试智能体")。
- **智能体 ID**: 智能体的唯一数字标识符，位于页面的 URL 中 (例如, "33")。
- **描述**: 智能体用途或特征的简要描述 (例如, "这是一个用于文档的测试智能体")。

## 可用操作

在仪表盘上，您通常可以执行以下操作：

- **编辑**: 允许您点击各操作按钮，进入各模块对智能体进行配置和编辑。

此页面可作为快速参考点，用于在深入了解更详细的配置之前识别您的智能体并了解其基本状态。
