# 手势（待开发）

“手势”模块用于配置和测试手势识别功能。

## 配置选项

### 预定义手势

- 系统内置多种常用手势，可直接调用，包括：
    - **挥手**：标准的挥手动作。
    - **竖起大拇指**：表达赞同或积极。
    - **指向**：用于引导用户注意力。
    - **鼓掌**：用于喝彩或表达热情。
- 每个手势均可进行如下操作：
    - **校准**：启动该手势的校准流程，确保动作准确、流畅。
    - **重置**：恢复该手势至默认状态。

### 自定义手势

- 支持创建或导入自定义手势（如平台已开放该功能），可在本节进行管理。

合理配置和使用手势，有助于提升智能体的非语言交流能力，使人机交互更加生动、直观。
