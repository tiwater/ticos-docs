# 语音

“语音”模块用于配置智能体的发声方式和语音参数。

## 配置选项

### 使用自定义语音 ID

- 启用后，可为智能体指定自定义语音（如克隆语音或特殊配置语音）。启用后将显示输入框，用于填写自定义语音 ID。

### 语音选择

- 通过下拉菜单选择智能体的发声风格，包括性别、口音和语言等。
- 示例：如 “Alice (英语, 美国, 女性)”、“普通话, 女性”。

### 语速

- 通过滑块调整语音播报速度。

### 音高

- 通过滑块调整发声音高，可正负调节。

### 音量

- 通过滑块调整语音输出的响度。

### 测试语音

- 输入文本，点击“测试语音”按钮，可实时试听当前语音设置的效果。

通过合理配置上述参数，可使智能体语音更具个性和亲和力，提升用户交互体验。特性与其角色和您的品牌形象一致的代理至关重要。
