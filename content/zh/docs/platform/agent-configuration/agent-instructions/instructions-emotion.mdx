# 情绪 (反应)

“情绪”设置位于“指令”->“反应”模块下，允许您配置您的具身智能体如何表达情绪。这可以显著影响智能体给人的个性印象和用户参与度。

## 配置选项卡

情绪配置页面通常分为两个主要选项卡：

### 1. 情绪强度

- **描述**：此选项卡（例如，在探索中默认选中，`ref=e90`）专注于控制智能体情绪表达的强度或细微程度。
- **主要控件**（基于探索）：
    - **情绪强度下拉菜单/选择**：（例如 `ref=e95`）允许您设置情绪显示的总体强度级别。选项可能包括“细微”、“正常”、“夸张”或数字刻度。
    - **面部表情按钮/链接**：（例如 `ref=e105`）可能链接到更详细的特定面部表情（与情绪相关）配置区域。
    - **搜索框**：（例如 `ref=e101`）如果列表内容广泛，可用于查找特定的情绪或表情设置。

### 2. 情绪过渡

- **描述**：此选项卡（例如 `ref=e102`）可能处理智能体如何从一种情绪状态转变为另一种状态。平滑自然的过渡是实现可信情绪表达的关键。
- **主要控件**（基于探索，尽管此选项卡的详细信息未完全捕获）：
    - **过渡速度/风格**：用于控制过渡速度或方式的设置（例如，渐进式、突变式）。
    - **情绪映射**：可能是关于某些事件或输入如何触发情绪状态变化的规则或映射。
    - **搜索框**：与“强度”选项卡类似，可能存在搜索框，用于导航复杂的过渡规则。

## 通用功能

- **目的**：本部分的总体目标是允许对智能体的情绪表现进行微调。这可以使交互感觉更自然、更具同理心，或更符合智能体定义的角色。
- **注意事项**：智能体可以显示的特定情绪以及控制的粒度将取决于底层虚拟形象和人工智能模型的能力。

恰当地配置情绪反应可以使您的智能体更具亲和力，并改善整体用户体验。然而，重要的是要确保情绪表达与智能体的角色和交互上下文保持一致。
