# 技能

“技能”模块允许您为智能体添加并管理用于执行特定任务、与外部系统交互或处理高级信息的能力。这些技能本质上是智能体可以调用的工具或函数。

## 总览

“技能”页面包含以下几个核心部分：

- **页面标题**：“技能”。
- **内容区域**：此区域将列出已添加到智能体的所有技能。如果尚未添加任何技能，则会显示相应的提示信息。
- **主要操作**：用于添加新技能的按钮。

## 添加技能

您可以通过以下几种方式为智能体添加新技能：

1.  **创建新技能**
    -   **功能**：点击此按钮将打开一个界面，让您可以从头开始定义一个新技能。您需要指定技能的名称、描述、输入/输出参数，并编写其业务逻辑或将其连接到指定的 API 端点。

2.  **从库中添加**
    -   **功能**：此选项允许您访问一个预置的技能库。您可以浏览或搜索此库，选择平台提供的或由社区共享的通用技能，并将其直接添加到您的智能体中。

3.  **从 MCP 服务器添加**
    -   **功能**：MCP（模型上下文协议）服务器通过提供对专门功能或外部信息的访问来扩展智能体能力。此选项允许您将智能体连接到一个 MCP 服务器，并集成其上托管的技能。

## 管理现有技能

技能被添加到智能体后，您可以在此页面进行管理：

-   查看已配置技能的详细信息。
-   配置单个技能的参数或设置。
-   移除不再需要的技能。

为您的智能体配备正确的技能组合，是使其能够执行广泛任务并与用户和系统进行有效交互的关键。
