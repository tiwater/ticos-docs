# 知识库

“知识库”模块允许您管理您的具身智能体用于提取信息、回答问题和执行任务的知识库。一个精心策划的知识库对于创建一个智能且有用的智能体至关重要。

## 总览

当您导航到代理的知识库部分时，通常会看到以下信息：

- **页面标题**：“知识库”
- **记忆功能**：开启知识库的记忆功能。
- **脚本功能**：为智能体配置对话脚本。

## 记忆功能

配置智能体的记忆功能。

- **启用记忆功能**: 启用后，知识库将转化为智能体的长期记忆，提供更全面的交互体验。终端级的知识库的上传和管理，请参见相关 API 说明。

## 脚本

要向智能体的脚本库中添加新信息，请使用“添加脚本”功能。当用户的对话包含脚本库中的脚本时，智能体将能够根据脚本内容生成回答。

1.  **点击“添加脚本”按钮**。
    此操作将打开一个标题为“添加脚本”的对话框。

2.  **填写知识详情**：
    对话框通常包含以下字段：

    -   **名称**：
        -   **描述**：您要添加的知识项或文档的简洁描述性名称。
        -   **示例**：“2024年第一季度产品规格”

    -   **描述**：
        -   **描述**：此知识项内容或用途的更详细摘要。
        -   **示例**：“包含2024年第一季度发布的所有产品的详细规格，包括尺寸、材料和功能。”

    -   **标签**：
        -   **描述**：有助于对此知识项进行分类和搜索的关键字或标签。根据系统的实现，标签可以用逗号或空格分隔。
        -   **示例**：“产品, 规格, Q1, 2024, 硬件”

3.  **提交知识项**：
    对话框将有几个操作按钮：

    -   **取消按钮**：关闭对话框而不保存任何更改。
    -   **添加脚本按钮**：将新的知识项提交到智能体的知识库。
    -   **关闭按钮**：通常是对话框顶角的“X”图标，用于关闭它。

## 管理现有脚本

添加脚本项后，此部分通常还会提供以下工具：

-   列出并查看现有的脚本项。
-   编辑或更新现有项目。
-   删除不再相关的项目。

建立一个全面而准确的知识库是一个持续的过程，它能显著增强您的智能体的能力。
