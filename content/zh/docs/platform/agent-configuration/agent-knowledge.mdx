# 知识库

“知识库”模块允许您管理您的具身智能体用于提取信息、回答问题和执行任务的知识库。一个精心策划的知识库对于创建一个智能且有用的智能体至关重要。

## 总览

当您导航到代理的知识库部分时，通常会看到：

- **页面标题**：“知识库”
- **内容区域**：如果尚未添加任何知识，您可能会看到类似“尚未添加任何知识”和“添加知识以使您的智能体更智能”的消息。
- **主要操作**：“添加知识”按钮。

## 添加知识

要向智能体的知识库中添加新信息，请使用“添加知识”功能。

1.  **点击“添加知识”按钮**（例如，探索时的 `ref=e150`）。
    此操作将打开一个标题为“添加知识”的对话框（例如 `ref=e157`）。

2.  **填写知识详情**：
    对话框通常包含以下字段：

    -   **名称**（例如，文本框 `ref=e162`）：
        -   **描述**：您要添加的知识项或文档的简洁描述性名称。
        -   **示例**：“2024年第一季度产品规格”

    -   **描述**（例如，文本框 `ref=e165`）：
        -   **描述**：此知识项内容或用途的更详细摘要。
        -   **示例**：“包含2024年第一季度发布的所有产品的详细规格，包括尺寸、材料和功能。”

    -   **标签**（例如，文本框 `ref=e168`）：
        -   **描述**：有助于对此知识项进行分类和搜索的关键字或标签。根据系统的实现，标签可以用逗号或空格分隔。
        -   **示例**：“产品, 规格, Q1, 2024, 硬件”

3.  **提交知识项**：
    对话框将有几个操作按钮：

    -   **取消按钮**（例如 `ref=e170`）：关闭对话框而不保存任何更改。
    -   **添加知识按钮**（例如 `ref=e171`）：
        -   **功能**：将新的知识项提交到智能体的知识库。
        -   **关于启用状态的说明**：在探索过程中，观察到此按钮最初是禁用的。启用此按钮所需的具体条件（例如，填写所有字段，或一个未观察到的文件上传字段对特定内容格式的要求）尚不清楚，需要进一步调查 (参考 MEMORY[eb2b0bab-3efa-4e91-a558-c298f4c2d3ba])。
    -   **关闭按钮**（例如 `ref=e158`）：通常是对话框顶角的“X”图标，用于关闭它。

## 管理现有知识

添加知识项后，此部分通常还会提供以下工具：

-   列出并查看现有的知识项。
-   在知识库中搜索或筛选。
-   编辑或更新现有项目。
-   删除不再相关的项目。

（由于在初始会话期间未添加任何知识，因此未完全探索管理现有知识的具体用户界面。）

建立一个全面而准确的知识库是一个持续的过程，它能显著增强您的代理的能力。
