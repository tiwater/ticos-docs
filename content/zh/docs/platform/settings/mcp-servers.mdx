# MCP 服务

MCP（Model Context Protocol）服务是 Ticos 平台用于扩展具身智能体能力的核心机制。通过 MCP 服务，您的智能体可以与外部系统、第三方 API 或自定义的业务逻辑进行交互，从而实现超越其内置功能的复杂任务。

例如，您可以注册一个 MCP 服务来：
-   查询天气预报 API。
-   控制智能家居设备。
-   连接到公司的内部知识库。
-   执行自定义的计算任务。

## 主要功能

您可以在本模块进行以下操作：

-   **查看服务列表**：展示所有已注册的 MCP 服务，包括其名称、描述、端点地址和状态。
-   **注册新服务**：添加一个新的 MCP 服务。您需要提供服务的名称、描述、API 端点 URL 以及必要的认证信息（如 API 密钥）。
-   **编辑服务配置**：修改已注册服务的详细信息。
-   **删除服务**：永久移除一个 MCP 服务。

## 支持情况

- 服务器端 Stardust 模型在 realtime 模式下支持 SSE MCP 服务。
- 服务器端 OpenAI 的 gpt 模型在 agent responses 模式下支持 SSE 和 Streamable HTTP 两种方式的 MCP 服务。
