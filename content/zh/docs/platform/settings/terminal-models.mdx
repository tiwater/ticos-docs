# 终端型号

“终端型号”页面旨在提供一个受官方支持和预配置的终端硬件设备列表。用户可以在这里查看不同型号的终端的技术规格、功能特性以及适用的场景，以便为自己的应用选择最合适的硬件。

## 主要功能

一个功能完善的“终端型号”页面通常会包含以下信息：

-   **型号列表**：展示所有兼容的终端设备型号，包含设备图片、名称和关键特性摘要。
-   **详细规格**：点击某个具体型号后，可以查看其详细的技术参数，例如处理器、内存、摄像头规格、传感器类型、连接方式等。
-   **购买链接**：提供官方或推荐的购买渠道链接。
-   **设置指南**：提供针对特定型号的快速入门和配置文档链接。
