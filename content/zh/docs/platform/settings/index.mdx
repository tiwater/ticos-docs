# 设置总览

“设置”模块是您管理 Ticos 平台全局配置的中心。这些设置通常不针对单个具身智能体，而是应用于您账户下的所有终端和智能体，或定义了可供它们使用的资源。

您可以使用左侧的子菜单在不同的设置页面之间导航。

## 设置模块

以下是“设置”模块中可用的主要配置页面：

-   **[终端型号](./terminal-models.mdx)**：查看和管理平台支持的终端设备型号。这为您选择和配置硬件终端提供了参考。

-   **[人脸管理](./faces.mdx)**：管理您账户下所有终端的人脸识别数据库。您可以上传、命名和删除人脸，以便智能体能够识别特定的人物。

-   **[MCP 服务](./mcp-servers.mdx)**：配置和管理 MCP（Model Context Protocol）服务。MCP 服务允许智能体与外部系统或 API 进行交互，从而扩展其能力。
