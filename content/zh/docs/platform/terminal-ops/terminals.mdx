# 终端管理

终端管理功能允许您查看和管理所有已注册到 Ticos 平台的终端设备。通过这个界面，您可以监控终端状态、更新配置并执行各种管理操作。

## 功能概述

终端管理页面提供以下主要功能：

1. 查看所有终端设备的列表
2. 按设备名称或 ID 筛选终端
3. 查看终端详细信息
4. 更新终端配置
5. 监控终端状态和健康情况

## 界面元素

终端管理界面包含以下主要元素：

### 终端列表

显示所有已注册终端的列表，包括以下信息：
- 终端 ID
- 终端名称
- 终端类型
- 状态（激活/未激活等）

### 过滤和搜索

提供按以下条件筛选终端的功能：
- 关键字搜索

### 终端详情

选择特定终端后，可以查看如下详细信息（需对接终端数据接口）：

- **终端ID**
- **型号**
- **状态**
- **密钥**
- **位置**
- **最后更新**
- **电池电量**
- **运行时间**
- **软件版本**
- **分组**

## 管理操作

终端管理页面允许执行以下管理操作：

### 创建终端

- 创建单台终端设备
- 批量创建终端设备
- 管理终端设备批次

### 终端信息维护

- 更新终端信息（名称、描述）

### 删除终端

- 删除不再需要的终端设备

## 最佳实践

- 定期检查终端状态，确保所有终端正常运行

## 注意事项

- 某些配置更改可能需要重启终端才能生效
