# 终端运维总览

终端运维模块提供了一套全面的工具，用于管理和监控 Ticos 平台上的终端设备。通过这些工具，管理员可以实时查看终端状态、监控终端活动并管理终端配置。

## 主要功能

终端运维模块包含以下主要功能区域：

### [终端管理](./terminals.mdx)

“终端”页面允许您查看和管理所有已注册到您账户的终端设备。主要功能包括：

-   查看终端列表及其详细信息（如 ID、名称、状态）。
-   注册新终端。
-   编辑现有终端的配置。
-   删除不再使用的终端。

### [监控](./monitoring.mdx)

“监控”页面提供实时监控终端活动的能力，包括：

-   选择特定的“组”和“终端”进行监控。
-   实时查看所选终端与用户的对话内容。
-   访问高级设置以进行更精细的监控配置。
-   选择“记住选择”以便将来快速访问。

## 使用场景

终端运维模块适用于以下场景：

1. **终端设备管理**：集中管理所有终端设备
2. **问题诊断**：识别并解决终端设备问题
3. **性能监控**：监控终端设备的性能和使用情况
4. **用户体验分析**：分析终端与用户的交互情况

## 注意事项

- 访问终端运维功能需要适当的管理权限
- 某些高级功能可能需要额外的配置
- 监控功能应在遵守隐私政策和相关法规的前提下使用
