# 终端监控

终端监控功能允许管理员实时查看终端设备与用户之间的交互。通过这个功能，您可以监控对话内容、系统事件和终端性能，以便进行故障排除、质量控制和用户体验优化。

## 功能概述

终端监控页面提供以下主要功能：

1. 实时监控终端与用户的对话
2. 查看系统事件和日志
3. 监控终端性能指标
4. 配置高级监控选项

## 使用流程

1.  **选择组**：首先，从“选择组”下拉菜单中选择一个包含您想监控的终端的组。

2.  **选择终端**：选择组后，“选择终端”下拉菜单将被激活。从中选择您想监控的具体终端设备。

3.  **高级设置 (可选)**：点击“高级设置”按钮，可以弹出编辑框，用于配置服务器地址、端口等，具体信息请联系技术支持。

4.  **建立连接**：完成选择后，点击“连接”按钮，系统将建立与终端的连接，并开始在页面上显示实时的交互数据。

5.  **记住选择 (可选)**：如果您希望浏览器记住您当前选择的组和终端，可以勾选“记住选择”复选框。这样，下次访问该页面时，系统会自动填入这些信息。

## 监控界面

成功连接后，监控界面通常会分为几个区域：

-   **对话流**：实时显示用户与智能体之间的对话文本或语音转录内容。
-   **系统日志**：展示终端产生的系统级事件、错误信息或调试日志。
-   **视频流**：展示终端产生的视频流。
- **终端选择按钮**：用于选择终端的监控连接

## 最佳实践

- 仅在必要时进行监控，尊重用户隐私
- 将监控数据与用户反馈结合，全面评估用户体验

## 注意事项

- 监控功能应在遵守隐私政策和相关法规的前提下使用
- 高强度监控可能会对终端性能产生轻微影响
- 敏感信息应妥善处理，避免未授权访问
- 建议在监控前通知用户，获取适当的同意
