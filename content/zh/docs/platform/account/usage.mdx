# 使用情况

“使用情况”页面用于查看您在 Ticos 平台的 API 调用和 Token 消耗等统计数据，帮助您跟踪资源使用、分析趋势和优化应用。

## 访问使用情况页面

要访问使用情况页面：

1.  点击平台界面右上角的用户头像或个人资料图标。
2.  从下拉菜单中选择“使用情况”。

## 主要内容

页面分为两个部分：

### 1. 指定时间段内的统计摘要

- **日期范围**：可选择指定时间段（最长支持 1 年），如 `2025-07-01 ~ 2025-07-15`。
- **总请求数**：该时间段内的 API 请求总数（如 524）。
  - **每日均值**：平均每天的请求数（如 ~58 每天）。
- **总 Token 数**：该时间段内消耗的总 Token 数（如 94,142）。
  - **每次请求平均 Token**：平均每次请求消耗的 Token 数（如 ~180）。
- **活跃周期**：统计区间内有请求的天数（如 9 天，自 2025/7/2 起）。
- **提示 Token 数**：用于 prompt 的 Token 数及占比（如 24,500，26%）。
- **生成 Token 数**：用于生成回复的 Token 数及占比（如 69,642，74%）。
- **最近活动**：最近一次请求的时间（如 2025/7/15 12:52:06）。

### 2. 明细数据图表

- **日期范围**：可选择指定时间段（最长支持 1 个月），如 `2025-07-01 ~ 2025-07-31`。
- **请求数柱状图**：展示所选时间段内每天的请求数变化趋势。
- **Token 数柱状图**：展示所选时间段内每天的 Token 消耗趋势。
- 图表下方显示日期刻度，左边是数值坐标。

## 注意事项

- 统计数据支持最长 1 个月的时间段，图表按天聚合。
- Token 数包括 prompt 和生成两部分，并分别显示占比。
- 数据可能存在一定延迟。
- 如需更详细的账单或用量明细，请联系平台支持。

## 理解您的使用情况

定期查看您的使用统计可以帮助您：

-   **监控成本**：如果您的使用情况与定价计划相关联，请跟踪您的支出。
-   **识别趋势**：了解您的应用程序对 Ticos 服务的使用如何随时间变化。
-   **优化性能**：找出频繁使用或资源密集型的 API 调用，以便进行优化。
-   **容量规划**：根据历史使用模式预测未来需求。

## 注意事项

-   可用的具体指标可能取决于您的订阅级别和您在 Ticos 平台上使用的服务。
-   报告使用数据可能存在延迟，因此显示的统计信息可能并非精确到秒的实时数据。
-   如果您对使用情况或账单有疑问，请参阅平台的相关页面或联系支持。
