# 个人资料

个人资料页面用于管理您与 Ticos 身份相关的个人账户信息、偏好设置等。

## 访问您的个人资料页面

要访问您的个人资料页面：

1.  点击平台界面右上角的用户头像或个人资料图标。
2.  从下拉菜单中选择“个人资料”。

## 主要功能

-   **个人信息**：
    -   查看和编辑您的显示名称。
    -   查看与您账户关联的电子邮件地址。
    -   更新您的个人头像图片。
-   **密码管理**：
    -   更改您当前的密码。
-   **账户详情**：
    -   查看您的账户创建日期。
