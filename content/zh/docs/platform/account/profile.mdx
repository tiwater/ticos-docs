# 个人资料

个人资料页面用于管理您与 Ticos 身份相关的个人账户信息、偏好设置等。

## 访问您的个人资料页面

要访问您的个人资料页面：

1.  点击平台界面右上角的用户头像或个人资料图标。
2.  从下拉菜单中选择“账户设置”或类似选项。
3.  在账户设置中，导航至“个人资料”部分（例如，通过侧边栏链接 `ref=e49`）。

## 主要功能（预期）

一个典型的个人资料页面允许您管理账户的各个方面，可能包括：

-   **个人信息**：
    -   查看和编辑您的显示名称。
    -   查看与您账户关联的电子邮件地址。
    -   更新您的个人头像图片。
-   **密码管理**：
    -   更改您当前的密码。
-   **偏好设置**：
    -   设置平台界面的语言偏好。
    -   管理通知设置（例如，关于平台更新或警报的电子邮件通知）。
-   **账户详情**：
    -   查看您的用户 ID 或账户 ID。
    -   查看您的账户创建日期。
    -   管理任何已连接的第三方账户链接（例如，GitHub、Google）。
-   **安全设置**：
    -   启用或管理双因素认证（2FA）。
    -   查看活动会话并从其他设备注销。
