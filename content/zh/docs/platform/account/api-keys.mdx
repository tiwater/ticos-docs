# API 密钥

API 密钥用于以编程方式访问 Ticos 平台服务。每个账户可生成多个 API 密钥，适用于应用集成、自动化脚本等场景。

## 访问 API 密钥页面

1. 点击平台界面右上角的用户头像或个人资料图标。
2. 从下拉菜单中选择“API 密钥”。

## 功能说明

### 创建 API 密钥

- 点击“创建新API密钥”按钮，输入密钥名称后生成密钥。

### 管理 API 密钥

- “您的API密钥”区域列出所有已生成的密钥，包括：
  - 名称
  - 密钥片段（可显示或拷贝密钥供后续使用）
  - 创建日期
  - 操作（撤销/删除、编辑名称）

- 撤销或删除密钥后，该密钥将无法再用于 API 调用，操作不可恢复。

## 安全须知

- 妥善保管 API 密钥，避免泄露。
- 不要将密钥写入客户端代码或提交至版本管理系统。
- 推荐通过环境变量或安全密钥管理服务存储密钥。
- 为不同用途分别创建独立密钥，定期检查并撤销不再使用的密钥。
