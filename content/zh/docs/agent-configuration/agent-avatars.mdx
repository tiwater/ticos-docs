# 代理虚拟形象

代理配置中的“虚拟形象”部分旨在用于管理您的实体代理的视觉外观和角色。虚拟形象是您的代理与用户交互的图形表示。

## 导航和功能

- **链接到全局设置**：从特定于代理的配置菜单中，“虚拟形象”链接通常会重定向到全局设置页面，例如 `/en/settings/avatars` 或 `/en/settings/terminal-avatars`。
- **目的**：理想情况下，此部分将允许您：
    - 从可用的虚拟形象库中选择。
    - 自定义虚拟形象外观（例如，服装、特征、颜色）。
    - 管理特定于虚拟形象的动画或表情（尽管详细的情感/姿态表情通常在其各自的“反应”子模块中处理）。
