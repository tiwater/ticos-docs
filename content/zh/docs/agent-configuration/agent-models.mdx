# 代理模型

代理配置中的“模型”部分旨在用于管理驱动您的实体代理各种能力（例如语言理解、生成、视觉和语音）的底层人工智能 (AI) 模型。

## 导航和功能

- **链接到全局设置**：从特定于代理的配置菜单中，“模型”链接通常会重定向到全局设置页面，例如 `/en/settings/terminal-models`。
- **目的**：理想情况下，此部分将允许您：
    - 为不同的代理功能选择或分配特定的 AI 模型（例如，用于对话的语言模型，用于图像处理的视觉模型）。
    - 管理模型版本或在不同的可用模型之间切换。
    - 如果适用，配置特定于模型的参数。
    - 查看代理当前使用的模型的相关信息。

## 当前状态
