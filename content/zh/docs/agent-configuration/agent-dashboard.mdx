# 代理仪表盘 (总览)

代理仪表盘，也称为总览页面，是您选择代理后看到的第一个屏幕。它提供了代理基本信息的摘要以及对基本操作的快速访问。

## 显示的信息

仪表盘通常显示所选代理的以下详细信息：

- **代理名称**: 用户定义的代理名称 (例如, "测试代理")。
- **代理 ID**: 代理的唯一数字标识符 (例如, "33")。
- **描述**: 代理用途或特征的简要描述 (例如, "这是一个用于文档的测试代理")。
- **创建时间**: 代理最初创建的日期和时间 (例如, "2024-05-14 03:20:13")。
- **最后更新时间**: 代理配置最后修改的日期和时间 (例如, "2024-05-14 03:20:13")。

## 可用操作

在仪表盘上，您通常可以执行以下操作：

- **编辑**: 允许您修改代理的名称和描述。点击此按钮通常会打开一个类似于“创建新代理”的对话框。
- **删除**: 永久移除代理及其所有相关数据。此操作通常是不可逆的，并且可能需要确认。

此页面可作为快速参考点，用于在深入了解更详细的配置之前识别您的代理并了解其基本状态。
