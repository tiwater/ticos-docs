# 智能体仪表盘 (总览)

智能体仪表盘，也称为总览页面，是您选择智能体后看到的第一个屏幕。它提供了智能体基本信息的摘要以及对基本操作的快速访问。

## 显示的信息

仪表盘通常显示所选智能体的以下详细信息：

- **智能体名称**: 用户定义的智能体名称 (例如, "测试智能体")。
- **智能体 ID**: 智能体的唯一数字标识符 (例如, "33")。
- **描述**: 智能体用途或特征的简要描述 (例如, "这是一个用于文档的测试智能体")。
- **创建时间**: 智能体最初创建的日期和时间 (例如, "2024-05-14 03:20:13")。
- **最后更新时间**: 智能体配置最后修改的日期和时间 (例如, "2024-05-14 03:20:13")。

## 可用操作

在仪表盘上，您通常可以执行以下操作：

- **编辑**: 允许您修改智能体的名称和描述。点击此按钮通常会打开一个类似于"创建新智能体"的对话框。
- **删除**: 永久移除智能体及其所有相关数据。此操作通常是不可逆的，并且可能需要确认。

此页面可作为快速参考点，用于在深入了解更详细的配置之前识别您的智能体并了解其基本状态。
