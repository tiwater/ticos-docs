# 高级设置

“高级设置”模块提供对与您的具身智能体管理相关的关键、通常具有破坏性的操作的访问。

## 总览

当您导航到“高级设置”页面（例如 `/en/agents/33/settings`）时，通常会看到：

- **页面标题**：“智能体设置”
- **副标题/描述**：“当前具身智能体的高级设置。”

## 主要功能和操作

### 1. 删除具身智能体

这是高级设置部分中通常可以找到的主要功能。

- **部分标题**：“删除具身智能体”（例如，探索时的 `ref=e155`）。
- **警告/描述**：通常会显示醒目的警告消息，强调此操作的永久性。例如：“此操作无法撤销。您的智能体及其所有数据将被永久删除。”（例如 `ref=e157`）。
- **删除智能体按钮**（例如，探索时的 `ref=e159`）：
    - **功能**：单击此按钮将启动永久删除当前智能体的过程。
    - **确认**：极有可能在单击此按钮后出现确认对话框，以防止意外删除。您需要确认删除智能体的意图。

## 重要注意事项

- **不可逆性**：删除智能体是永久性操作。所有相关的配置、知识库项目以及与该智能体明确关联的任何其他数据都将丢失且无法恢复。
- **谨慎操作**：使用此功能时请务必格外小心。在继续删除之前，请确保您已备份所有必要信息，或确定不再需要该智能体。

本节保留用于高影响操作。在高级设置中执行操作之前，请务必仔细检查您正在管理的智能体。
