# 核心指令

“核心指令”部分用于定义您的实体代理的基本操作准则和个性。这些指令是代理行为和响应的主要指导。

## 配置选项

### 1. 核心指令文本区域

- **描述**：这是一个大型文本字段，您可以在其中输入代理的主要指令集。这些指令可以规定其角色、个性特征、交互方式、应优先处理的信息以及必须遵守的任何特定约束或准则。
- **用法**：您可以编写自然语言指令。例如，您可以指示代理成为一个乐于助人的助手、特定主题的知识渊博的专家，或者采用特定的沟通风格（例如，正式、随意、幽默）。
- **示例文本**：`“你是一个为 Ticos 平台提供帮助的友好助手。你的目标是提供关于我们服务的准确信息，并引导用户完成常见任务。始终保持礼貌和专业的语气。”`

### 2. 回应语言

- **描述**：此设置允许您指定代理回应时应使用的主要语言。
- **控件**：通常是一个列出可用语言的下拉菜单。
- **示例**：如果您选择“英语（美国）”，代理将主要以美式英语回应。如果您选择“简体中文”，它将以简体中文回应。
- **注意**：代理理解和回应不同语言的能力可能取决于底层的人工智能模型及其多语言能力。

设置清晰全面的核心指令对于确保您的代理按预期行事并有效实现其预定目标至关重要。
