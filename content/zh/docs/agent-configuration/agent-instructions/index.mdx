# 智能体指令

“指令”模块是配置您的具身智能体的关键部分。它允许您定义智能体如何理解其核心目标、感知环境以及制定反应。

该模块通常分为几个子部分，每个子部分侧重于智能体操作逻辑的特定方面：

- **[核心指令](./instructions-core.mdx)**：定义智能体的基本准则、个性及主要目标。
- **感知**：配置智能体如何解读感官输入。
    - **[视觉](./instructions-vision.mdx)**：与视觉输入、物体识别和细节级别相关的设置。
    - **[听觉](./instructions-hearing.mdx)**：与音频输入和语音检测相关的设置。
- **反应**：决定智能体如何回应和表达自己。
    - **[语音](./instructions-speech.mdx)**：配置声音特征、语音模式和语言。
    - **[情绪](./instructions-emotion.mdx)**：定义情绪反应和表达强度。
    - **[姿态](./instructions-gesture.mdx)**：管理身体姿态和空闲动作。
    - **[行为](./instructions-behavior.mdx)**：*（注意：此部分目前正在审查中，因为在探索过程中无法访问。）* 用于定义更广泛的行为模式和反应。

这些领域的详细配置选项均在其各自的页面中介绍。有效的指令设置是创建行为一致且符合您预期应用的智能体的关键。
