# 语音 (反应)

“语音”设置位于“指令”->“反应”模块下，允许您自定义您的具身智能体如何发声回应以及如何使用语音进行交互。

## 配置选项

### 1. 使用自定义语音 ID

- **控件**：一个拨动开关（例如，探索时的 `ref=e89`）。
- **功能**：此选项可能启用为智能体使用特定的、预定义的自定义语音，该语音可能是经过克隆或特别配置的。启用后，它可能会显示一个用于输入自定义语音 ID 的额外输入字段。

### 2. 语音选择

- **标签**：语音（或类似名称）
- **控件**：一个下拉菜单（组合框，例如，探索时的 `ref=e96`）。
- **功能**：允许您从智能体可用的标准语音列表中进行选择。这些语音在性别、口音和语言方面可能有所不同。
- **示例**：“Alice (英语, 美国, 女性)”，“普通话, 女性”（实际名称可能会有所不同）。

### 3. 语速

- **标签**：语速（或类似名称）
- **控件**：一个滑块（例如，探索时的 `ref=e111`）。
- **功能**：调整智能体说话的速度。移动滑块通常会改变每分钟的字数。
- **范围**：数值范围可能从比标准语速慢（例如 0.5x）到快（例如 2.0x）。

### 4. 音高

- **标签**：音高（或类似名称）
- **控件**：一个滑块（例如，探索时的 `ref=e118`）。
- **功能**：修改智能体声音的音高，使其听起来更高或更低。
- **范围**：数值可以表示半音程的调整或相对音阶。

### 5. 音量

- **标签**：音量（或类似名称）
- **控件**：一个滑块（例如，探索时的 `ref=e125`）。
- **功能**：控制智能体语音输出的响度。

### 6. 测试语音

- **功能**：此功能允许您使用当前设置预览代理的声音。
- **组件**：
    - **文本输入字段**（例如 `ref=e134`）：一个文本框，您可以在其中键入示例文本供代理朗读。
    - **“测试语音”按钮**（例如 `ref=e130`）：点击此按钮将使代理使用选定的语音、语速、音高和音量设置来朗读输入字段中输入的文本。

仔细配置这些语音参数对于创建一个易于理解且声音特性与其角色和您的品牌形象一致的代理至关重要。
